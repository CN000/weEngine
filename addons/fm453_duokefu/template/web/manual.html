关于本模块
  1. 为了获得最好的后台操作体验，我们建议您使用最新版Goole浏览器。（火狐等部分浏览器不支持点击粘贴功能）
  2. 本模块扩展配置的多客服客户端插件仅适用于认证并启用了多客服功能的公众号（包括认证了的服务号与订阅号）。
  3. 尤其是初次安装使用的用户，请务必看“操作说明”
  4. 模块中大部分操作采用的是追加数据的方式；为了简化设计，我们对很多链接使用了在新窗口中打开的方式，以便于同步操作与保存。
  5. 关于模块的界面形式（无左侧的功能框）：作了界面调整，并非BUG；请放心使用。
  6. 查看更新说明
